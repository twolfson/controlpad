# controlpad

WindowPad functionality for Linux.

Built on top of [wmctrl][wmctrl] and [xwininfo][xwininfo].

[wmctrl]: http://tomas.styblo.name/wmctrl/
[xwininfo]: http://www.xfree86.org/4.2.0/xwininfo.1.html
